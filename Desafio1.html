Estrutura básica do HTML

O HTML são textos delimitados por nomes que o navegador consegue interpretar e cada nome destes limitadores, tem uma função e comportamento específico, dito isso, aqui está as 3 estruturas básicas do HTML:

  • HTML: Usado no começo e no fim do site, para o navegar entender o que será exibido.
  • HEAD: São coisa que ele carregar antes de exibir para o usúario, (como fontes, links adaptações no navegador e etc).
  • BODY: Usado para escrever o corpo do nosso site, tudo o que estiver dentro o navegador lê e interpreta para o usúario.

Falando sobre Tags

As tags HTML são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

  • I: Italico
  • U: Slinhado
  • MARK: Marca Texto
  • STRONG: Negríto
  • IMG: Colocar Imagens

Falando sobre Atributos

Atributos são comandos ou informações específicas que damos a uma tag, para que ela faça tal comando, porém com algum detalhe ou variação específica, em relação a tag original sem atributos.

  • ID="": Usado para identificar um elemento que você quer mexer/modificar/link.
  • STYLE="": Usada para formatar o estilo do seu texto, porêm, é mais usado no CSS.
  • CLASS="": Usada para padronizar um elemento dentro das ""(áspas), tambem usado no CSS.
  • TYPE="": Usado para o usúario escrever palavras ou colocar números.
  • SCR-"": Usado para exibir links, geralmente dentro da tag IMG.

Falando sobre Links

O link HTML, nada mais é que uma função do HTML que permite inserir os hiperlinks em diversos elementos, como textos e imagens. Um link precisa sempre apontar para uma URL (endereço) existente em seu site.

No HTML, os links são definidos pela tag (a)0. Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link. Dentro do conteúdo da tag (a), incluímos então o texto ou elemento que servirá como redirecionador, ou seja, que ao ser clicado, executará a função de redirecionar para o endereço dentro do atributo href.

Dessa forma, a sintaxe básica do HTML link é:

  • (a href="url">Exemplo)(/a)
